# 🚀 ArsyNox Labs — Portfolio

A **high-performance, modern portfolio website** for **ArsyNox Labs**, showcasing experimental web tools and services with a sleek **dark UI**, **glassmorphism**, and **bento-style layout**.

Built with **Vite + React + Tailwind CSS + Framer Motion**, optimized for **Cloudflare Pages**.

---

## ✨ Features

* 🌙 **Dark Mode UI** (default)
* 🧊 **Glassmorphism cards**
* 📦 **Bento Box project grid**
* 🎞 **Smooth animations** using Framer Motion
* ⚡ **Lightning-fast** (Vite + static build)
* 📱 Fully **responsive** (mobile → desktop)
* 🖼 **Logo & favicon support**
* ☁️ **Cloudflare Pages ready**

---

## 🛠 Tech Stack

* **React 18**
* **Vite**
* **Tailwind CSS**
* **Framer Motion**
* **Lucide Icons**
* **TypeScript**

## 🚀 Getting Started (Local Development)

### 1️⃣ Clone the repository

```bash
git clone https://github.com/your-username/arsynox-portfolio.git
cd arsynox-portfolio
```

### 2️⃣ Install dependencies

```bash
npm install
```

### 3️⃣ Start dev server

```bash
npm run dev
```

Open in browser:

```
http://localhost:5173
```

---

## 🖼 Logo & Favicon Setup

Place your files here:

```
public/
├── logo.png
└── favicon.png
```

Test directly:

```
http://localhost:5173/logo.png
http://localhost:5173/favicon.png
```

---

## ☁️ Deploy on Cloudflare Pages

**Build settings:**

| Option           | Value           |
| ---------------- | --------------- |
| Framework        | Vite            |
| Build Command    | `npm run build` |
| Output Directory | `dist`          |

After pushing to GitHub, Cloudflare Pages will auto-deploy 🚀
